这是一款运行在Chrome上的 python shell ，可以让您不必安装python环境，即可运行python脚本。当然也不仅限于Python，这个二十几兆的扩展还支持Ruby，JavaScript等等等等。

支持语言：


* Python 2.7
    * Ruby 1.8
    * JavaScript
